# v2.0.9

## Bug Fixes

### Fixed Event/Relationship Update Not Reflecting in UI
- **Issue**: When updating event or relationship data, changes don't appear until page refresh
- **Cause**: API returns data in `result.data` but code was looking for `result.event` or `result.relationship`
- **Solution**: Updated all API response handlers to use `result.data` consistently

### Changes Made
- Fixed `handleCreateEvent` to use `result.data` instead of `result.event`
- Fixed `handleUpdateEvent` to use `result.data` instead of `result.event`
- Fixed `createRelationship` to use `result.data` instead of `result.relationship`
- Fixed relationship dialog update to use `result.data` instead of `result.relationship`

### Files Changed
- `src/components/board/investigation-board.tsx` - Fixed all API response handlers
