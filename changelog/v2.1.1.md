# v2.1.1

## Bug Fixes

### Fixed Black Shadow Around Relationship Edges (Second Attempt)
- **Issue**: Black shadow still appearing around relationship edge lines
- **Cause**: SVG filter effects and marker shadows from React Flow default styles
- **Solution**: 
  - Added explicit `filter: 'none'` to the edge path style
  - Added comprehensive CSS overrides to remove all filter effects from React Flow edges
  - Removed any background paths or glow effects

### Changes Made
- Updated `relationship-edge.tsx` to explicitly set `filter: 'none'` on the path element
- Added more CSS rules to `globals.css` to override default React Flow edge styles:
  - Remove filters from `.react-flow__edge-path`
  - Remove filters from `.react-flow__arrowhead` (marker)
  - Remove filters from `.react-flow__edge-text`
  - Add `stroke-linecap: round` and `stroke-linejoin: round` for cleaner edges
  - Remove `backdrop-filter` effects

### Files Changed
- `src/components/board/relationship-edge.tsx` - Explicit filter:none on path
- `src/app/globals.css` - Comprehensive edge shadow removal
