# v2.1.2

## Bug Fixes

### Fixed Curved/Straight Line Option for Relationships
- **Issue**: The "Curved" toggle option in relationship settings had no effect - lines were always curved
- **Cause**: The edge component always used `getBezierPath` regardless of the `isCurved` setting
- **Solution**: Updated the edge component to use `getStraightPath` when `isCurved` is false

### Changes Made
- Imported `getStraightPath` from `@xyflow/react` in addition to `getBezierPath`
- Added conditional logic to select the appropriate path function based on `isCurved`
- Default behavior remains curved (true) for backwards compatibility

### Files Changed
- `src/components/board/relationship-edge.tsx` - Added conditional path rendering based on isCurved
