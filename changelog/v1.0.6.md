# v1.0.6 - Bug Fix: DB Manager Store Functions

## التاريخ | Date
2025-03-06

## الوصف | Description

### المشكلة | Problem
خطأ `checkConnection is not a function` عند تحميل صفحة Database Manager.

### السبب | Root Cause
صفحة `db-manager/page.tsx` تحاول استخدام الدوال مباشرة من الـ store:
```javascript
const { checkConnection, fetchTables, fetchMetrics } = useDatabaseStore();
```

لكن الدوال موجودة داخل كائن `actions`:
```javascript
actions: {
  checkConnection: () => {...},
  fetchTables: () => {...},
  fetchMetrics: () => {...}
}
```

### الحل | Solution
تصحيح طريقة الوصول للدوال من الـ store:

```javascript
const {
  tables,
  connectionStatus,
  isLoadingTables,
  metrics,
  actions
} = useDatabaseStore();

useEffect(() => {
  actions.checkConnection();
  actions.fetchTables();
  actions.fetchMetrics();
}, [actions]);
```
