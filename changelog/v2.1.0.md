# v2.1.0

## Bug Fixes

### Fixed Relationship Edge Visual Issues
- **Issue**: Annoying black shadows appearing around relationship edges
- **Cause**: Multiple shadow and filter effects (drop-shadow-md, shadow-sm, glow effect) were applied to the edge
- **Solution**: Simplified the edge component by removing unnecessary shadow effects

### Changes Made
- Removed `drop-shadow-md` filter from selected edges
- Removed `shadow-sm` and `backdrop-blur-sm` from edge labels
- Removed the glow effect path for selected edges
- Simplified the styling to use clean borders without shadows
- Added proper ring-offset for selected state

### Files Changed
- `src/components/board/relationship-edge.tsx` - Removed shadow effects and simplified styling
