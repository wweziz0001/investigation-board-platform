### 6. إصلاح المصادقة في admin/layout.tsx

**المشكلة:**
عند الانتقال إلى `/admin` بعد تسجيل الدخول، يتم إعادة التوجيه إلى صفحة تسجيل الدخول.

**السبب:**
الـ layout يتحقق من `isAuthenticated` قبل استدعاء `checkAuth()` لتحديث الحالة من الـ cookie.

**الحل:**
```diff
+ const [isChecking, setIsChecking] = useState(true);

+ useEffect(() => {
+   const verifyAuth = async () => {
+     setIsChecking(true);
+     await checkAuth();
+     setIsChecking(false);
+   };
+   verifyAuth();
+ }, [checkAuth]);

  useEffect(() => {
-   if (!isAuthenticated) {
+   if (!isChecking && !isAuthenticated) {
      router.push('/login');
    }
  }, [isChecking, isAuthenticated, user, router]);
```

**الملفات المتأثرة:**
- `src/app/admin/layout.tsx` - إضافة التحقق من المصادقة

**النتيجة:**
- ✅ البقاء في لوحة الإدارة بعد تسجيل الدخول
- ✅ التحقق من صلاحيات الأدمن
