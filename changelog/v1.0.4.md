## 🔧 الإصلاحات

### إصلاح عرض قاعدة البيانات في لوحة الإدارة

**المشكلة:**
قاعدة البيانات لا تظهر في واجهة Database Management وتظهر رسالة خطأ:
```
TypeError: Cannot mix BigInt and other types, use explicit conversions
```

**السبب:**
SQLite يعيد قيم `COUNT(*)` كـ `BigInt`، لكن الكود يحاول جمعها مع أرقام عادية.

**الملفات المتأثرة:**
- `src/app/api/admin/db/tables/route.ts` - تحويل BigInt إلى Number
- `src/app/api/admin/db/metrics/route.ts` - تحويل BigInt إلى Number

**التغييرات:**

```diff
// tables/route.ts
- const countResult = await db.$queryRawUnsafe<{ count: number }[]>(
+ const countResult = await db.$queryRawUnsafe<{ count: bigint }[]>(
- const rowCount = countResult[0]?.count || 0;
+ const rowCount = Number(countResult[0]?.count || 0);

// metrics/route.ts
- const dailyActivity = await db.$queryRaw<Array<{ date: string; count: number }>>`
+ const dailyActivity = await db.$queryRaw<Array<{ date: string; count: bigint }>>`
- count: d.count,
+ count: Number(d.count),
```
